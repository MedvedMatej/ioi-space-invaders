## Navodila za lokalno postavitev

Predpogoj:
- Node.js
- Git

V konzoli izvrši naslednje ukaze:

- `git clone https://github.com/MedvedMatej/ioi-space-invaders`
- `cd ioi-space-invaders`
- `npm i`
- `npm run dev`

Odpri [http://localhost:3000](http://localhost:3000) v brskalniku.
